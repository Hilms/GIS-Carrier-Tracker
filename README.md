# 🌊 Global Fishing Watch – Data Visualization Project

## 📌 Overview

This project analyzes vessel activity data from the **Global Fishing Watch carrier dataset**.  
It focuses on maritime events involving fishing and carrier vessels, including encounters, port visits, and loitering behavior.

The main goal is to explore **global fishing patterns**, **vessel movements**, and **regulatory compliance** using interactive data visualizations.

---

## 📊 Dataset

- **Source:** https://globalfishingwatch.org/data-download/datasets/carriers:v20220124  
- **Last updated:** 30.11.2022  
- **Provider:** Global Fishing Watch  

The dataset contains three main event types:

- 🚢 **Port events** – vessels docking or anchoring in ports  
- 🌊 **Loitering events** – vessels waiting in open waters  
- 🤝 **Encounter events** – interactions between carrier and fishing vessels  

Each record includes:
- Vessel identity and type
- Time range (start/end)
- Geographic position
- Speed and movement metrics
- RFMO (Regional Fisheries Management Organization) region data
- Authorization status of fishing activity

---

## 🎯 Objectives

- Identify fishing import/export patterns by country  
- Detect authorized vs. unauthorized fishing encounters  
- Analyze travel distances of carrier vessels  
- Measure port traffic over time  
- Compare global fishing activity across regions  

---

## 📈 Visualizations

This project includes multiple interactive visualizations:

- 🥧 **Country-level aggregation** (pie charts of vessel activity)  
- 🗺️ **Map visualization** of encounters (authorized vs. unauthorized)  
- 🔥 **Heatmap / pixel density maps** of fishing activity  
- 📊 **Average travel distance per country** (bar charts)  
- ⚓ **Port traffic analysis** (arrivals and departures over time)  

---

## ❓ Key Research Questions

- Which countries operate in which ocean regions?  
- Where do unauthorized encounters occur most frequently?  
- How far do fishing carrier vessels travel globally?  
- Which ports handle the highest traffic?  
- Are there correlations between fishing effort and trade routes?  

---

## 🚀 Goal

The project aims to provide an intuitive, data-driven understanding of global fishing activity and highlight patterns in maritime logistics, compliance, and regional fishing behavior.
